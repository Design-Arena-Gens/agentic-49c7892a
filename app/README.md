## Parking Command Center

Tenaent & Guest parking approvals with instant confirmation dispatch.

### Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`.

### Core Workflows

- Register tenants or guests (email + phone required)
- Assign primary or secondary vehicle slots per account (max two vehicles)
- Approve parking windows with hour-based allotments and internal notes
- Track vehicle lifecycle (approved → parked → completed)
- Monitor the confirmation dispatch log for email/SMS notifications
